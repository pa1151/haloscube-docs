Updates

Halos Cube supports Over the Air (OTA) Updates. Updates provide new features or address issues with the software.

  1. Open dashboard: http://halos-cube.local
  2. Click "Updates" ‚Üí "Check for Updates" button
  3. If the Latest version is newer, click "Install Update"
  4. Wait for the firmware to update (<2 minutes)
    • While installing, do not power off!
    • When the update completes, the cube will reboot.
  5. Wait for the cube to restart

Need Help?

Nothing happens when I power on

  • Check that the cube is getting power
  • Try a different USB power supply
  • Display will show splash screen briefly on startup

Can't see "Halos-Cube-Setup" network

  • Wait 60 seconds after power on
  • Check you're looking at 2.4GHz networks (Halos Cube does not support 5GHz)
  • Cube might already be connected to WiFi - check if clock is showing
  • Press the Reset Button (Front on right side) and check again

Can't connect to the setup network

  • Forget the network and try again
  • Move closer to the cube

"http://halos-cube.local" doesn't work

  • Ensure that your phone/computer is on the same WiFi network

Display shows weird colors or glitches

  • Unplug and replug the cube
  • May need to update firmware - Dashboard ‚Üí Updates

How do I change or reset my network?

  1. Open dashboard: http://halos-cube.local
  2. Click "Advanced" ‚Üí "Reset WiFi Settings" button
  3. Cube reboots into setup mode
  4. Start over from Step 2

What if an update fails

  • In the event of a failed update, the Cube will continue to use the existing version
  • Try again in a few minutes

How do I perform a Factory Reset

  1. Open dashboard: http://halos-cube.local
  2. Click "Advanced" ‚Üí "Factory Reset" button
